## 场景一：暧昧期聊天

> **情感特征**: 小心试探、欲擒故纵、甜蜜期待

**时间**: 周五晚上 22:30

```
林小雨: 在吗？
陈默: 嗯，刚洗完澡
林小雨: 哦...
陈默: 怎么了？这么晚还没睡
林小雨: 睡不着
陈默: 有心事？
林小雨: 也没有啦，就是...想找人聊聊天
陈默: 那正好，我也睡不着
林小雨: 😊
陈默: 今天看你朋友圈发的那家店，好吃吗？
林小雨: 还不错诶！你要去吗？
陈默: 想去，但是一个人去有点奇怪
林小雨: 那...我陪你去？
陈默: 好啊，周末有空吗
林小雨: 看看吧～
陈默: 那就说定了？
林小雨: 嗯嗯 😊
陈默: 早点睡吧，明天还要上班
林小雨: 好，晚安
陈默: 晚安 🌙
林小雨: 对了...
陈默: 嗯？
林小雨: 没什么，晚安啦！
陈默: 晚安，做个好梦 💫
```

**时间**: 周六上午 10:15

```
陈默: 早安 ☀️
林小雨: 早～这么早就起了？
陈默: 想着今天要出门，就早点起来了
林小雨: 去哪呀？
陈默: 不是说好一起去那家店吗？
林小雨: 啊？我还没化妆呢！
陈默: 不急，我等你
林小雨: 那你先去忙别的吧，我大概要一个小时
陈默: 没事，我就在你家楼下等你
林小雨: 啊？？？你已经在了？
陈默: 嗯，买了早餐给你
林小雨: 你怎么不早说！！！我马上下来！！！
陈默: 别急，慢慢来 😊
```

## 场景三：吵架与和好

> **情感特征**: 情绪激动、委屈、反思、和解

**时间**: 周三晚上 20:00 (吵架中)

```
李梦瑶: 你什么意思？
张浩然: 什么什么意思
李梦瑶: 你今天为什么没来接我
张浩然: 我不是说了今天加班吗
李梦瑶: 你就不能早点走吗
张浩然: 项目明天要交，我怎么走？
李梦瑶: 那你就让我一个人在雨里等了半小时？
张浩然: 你可以打车啊
李梦瑶: 打车？你知道今天多难打车吗？我等了那么久你就这态度？
张浩然: 我已经在道歉了你还想怎样
李梦瑶: 你哪有道歉？你全程在找借口！
张浩然: 我没有找借口，我在解释
李梦瑶: 呵呵
张浩然: 你别这样
李梦瑶: 我怎样了？我等了你半小时，淋了雨，感冒了，你就这态度？
张浩然: 你感冒了？严重吗？
李梦瑶: 不用你管
张浩然: 我现在过去找你
李梦瑶: 不用
张浩然: 梦瑶
李梦瑶: 我不想理你
```

**时间**: 周三晚上 22:30 (和好)

```
张浩然: 开门
李梦瑶: 你怎么来了
张浩然: 给你买了药和姜茶
李梦瑶: ...
张浩然: 对不起，今天是我不好
李梦瑶: 你知道我等了多久吗
张浩然: 知道，是我的错，我应该提前跟你说清楚
李梦瑶: 我不是不理解你加班，我只是...只是很委屈
张浩然: 我知道，我以后一定提前跟你说，如果真的走不开，我给你打车
李梦瑶: 我也不是非要你接我
张浩然: 但我想接你，今天是真的走不开
李梦瑶: 那你刚才态度那么差
张浩然: 对不起，我当时也压力很大，语气不好
李梦瑶: 算了，我也有点任性
张浩然: 你没有，你等我那么久，委屈是正常的
李梦瑶: 哼
张浩然: 别哼了，先把药吃了
李梦瑶: 哦
张浩然: 还生气吗？
李梦瑶: 一点点
张浩然: 那我抱抱你？
李梦瑶: 不要
张浩然: 真的不要？
李梦瑶: ...那你抱吧
张浩然: ❤️
李梦瑶: 下次不许这样了
张浩然: 好，我保证
```

赵晓晨: 但是什么？你有别人了？
孙浩宇: 没有，你别乱想
赵晓晨: 那为什么突然说不合适？
孙浩宇: 不是突然，我想了很久
赵晓晨: 你想了很久都不跟我说？
孙浩宇: 我不知道怎么开口
赵晓晨: 你就这么想分手？
孙浩宇: 我觉得我们在一起...不快乐
赵晓晨: 我很快乐啊！
孙浩宇: 但我不快乐
赵晓晨: ...你为什么不快乐
孙浩宇: 你总是管我，我喘不过气
赵晓晨: 我管你？我关心你也有错吗？
孙浩宇: 你那是关心吗？你那是控制
赵晓晨: 我控制你？我只是想知道你在哪、在干嘛
孙浩宇: 你知道吗，我每次跟朋友出去你都要打电话

孙浩宇: 但你那样让我很没面子
赵晓晨: ...
孙浩宇: 我需要空间

孙浩宇: 晓晨，我们都冷静一下吧
